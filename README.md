# NYC Taxi Data Analysis 🚖

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on NYC Taxi dataset to find insights and patterns.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📊 Key Analysis
- Data cleaning
- Missing value handling
- Outlier detection
- Trend analysis

## 📁 Files
- EDA notebook (.ipynb)
- PDF report
